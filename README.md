# Starway Diecast Demo

Production-style demo storefront for `starwaystudio.com`, built with Next.js App Router, TypeScript, Tailwind CSS, and local mock data.

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open `http://localhost:3000`

## Included

- Home, shop, collections, brands, about, contact, cart, wishlist, FAQ, and product detail routes
- Mock catalog with 60 collector-oriented products
- URL-driven filtering, search, sorting, and load-more behavior
- Local cart, wishlist, and recently viewed state using `localStorage`
- Placeholder automotive product visuals ready to swap for real imagery later

## Notes

- This is a front-end-only demo. Checkout, contact form submission, and backend inventory are intentionally mocked.
- Product imagery is rendered as styled placeholders rather than real brand photos.
